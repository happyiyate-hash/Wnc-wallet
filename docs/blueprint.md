# **App Name**: Wevina v2

## Core Features:

- Wallet Generation: Allow users to generate a new crypto wallet with a unique address.
- Key Storage: Securely store private keys using encryption.
- Transaction Management: Enable users to send and receive cryptocurrencies, and track transaction history.
- Currency Conversion Tool: Tool to allow users to convert crypto currencies. An LLM tool decides on data validity before displaying the data
- Security Alerts: Implement alerts to notify users of any potential security threats or suspicious activity.

## Style Guidelines:

- Primary color: Deep violet (#673AB7) to convey trust and security.
- Background color: Light lavender (#F3E5F5) for a soft, uncluttered backdrop.
- Accent color: Soft pink (#E91E63) for highlights and important actions, analogous to violet.
- Body and headline font: 'Inter', a grotesque-style sans-serif known for its neutral and modern look.
- Use minimalist, line-based icons to maintain a clean and modern appearance.
- Employ a clear, intuitive layout that emphasizes ease of use and security features prominently.
- Subtle animations for transactions and loading states to enhance user experience without being distracting.
